JetBrains выпустила обновлённый менеджер Toolbox 2.0

4 часа назад

Прошлая версия Toolbox устанавливалась в пользовательском каталоге с постоянно меняющимся путём. Теперь каталог установки по умолчанию соответствует рекомендациям операционной системы. Toolbox 2.0 использует следующие пути:

  • %LocalAppData%\Programs для Windows;
  • ~/Applications для macOS;
  • ~/.local/share/JetBrains/Toolbox/apps для Linux.
